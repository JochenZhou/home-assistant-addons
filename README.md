# Home Assistant Add-ons by Jcohen

[![Add Repository](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FJochenZhou%2Fhome-assistant-addons)

## 一键安装

点击上方按钮，或手动添加仓库地址：

```
https://github.com/JochenZhou/home-assistant-addons
```

### 手动安装步骤

1. 在 Home Assistant 中，进入 **设置** > **加载项** > **加载项商店**
2. 点击右上角 **⋮** 菜单，选择 **仓库**
3. 添加仓库地址：`https://github.com/JochenZhou/home-assistant-addons`
4. 点击 **添加** > **关闭**
5. 刷新页面，即可看到本仓库的加载项

## 可用加载项

### [Sherpa-Onnx-STT-RKNPU](./sherpa_onnx_stt_rknpu)

基于 [sherpa-onnx](https://github.com/k2-fsa/sherpa-onnx) 的 RK3566/RK3588 NPU 加速语音识别服务。

- 支持中英日韩粤多语言
- RK3566/RK3588 NPU 硬件加速
- SenseVoice 模型情感识别
- Wyoming 协议支持，与 Home Assistant 无缝集成
